◆ Packed

template<typename TagT>
uint32_t Tactix::FTactixHandle< TagT >::Packed {0}

Packed payload, layout [high 16 = generation][low 16 = index].

Packed == 0 is reserved to mean "invalid", which is why valid pool slots always start their generation at 1 rather than 0. Without that reservation a real handle to slot 0 / generation 0 would be indistinguishable from a default-constructed empty handle.
