Detailed Description

Adapts a UE pawn to Tactix::ITactixAgent and holds its AI state.

The vital and tactical UPROPERTY fields are the source of truth that BuildContext snapshots each decision. Blueprint and BT tasks write to them (taking damage, spending ammo, entering cover); the AI only ever reads the snapshot.

FTactixAgentContext UTactixAgentComponent::BuildContext ( ) const
overridevirtual

Builds the agent snapshot from the current component state.

Converts the UE-side fields into a FTactixAgentContext: vitals become ratios, the owner's transform fills position/velocity/forward, and threat distance is derived from ThreatLocation.

Returns
The populated context for this frame.
