◆ Alloc()

void * Tactix::FTactixArena::Alloc ( std::size_t Size,
std::size_t Align )
inlinenoexcept

Carves Size bytes off the buffer at the requested alignment.

Rounds the cursor up to Align, then advances it by Size. Nothing is constructed; the bytes are raw.

Parameters
SizeNumber of bytes to reserve.
AlignRequired alignment in bytes. Zero is treated as 1. Must be a power of two; the round-up uses a bitmask, not a modulus.
Returns
Pointer to the reserved bytes, or nullptr if the remaining space (after alignment) can't fit Size.
