Detailed Description

The component that makes a pawn a Tactix agent.

Add this to any APawn that should take part in the Tactix AI systems: utility scoring, cover claiming, squad membership, influence stamping. It is the single bridge between Unreal actor space and the engine-agnostic Tactix::ITactixAgent interface. Once registered, every system in TactixCore and TactixSystems refers to this agent purely by its handle, never by an actor pointer, so the AI never reaches back into engine types.
