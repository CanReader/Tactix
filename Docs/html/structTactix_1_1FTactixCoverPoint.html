Detailed Description

One cover position registered with FTactixCoverSystem.

Normal points away from the wall into the open space the agent is exposed to. A threat counts as blocked when it sits roughly along -Normal from Position, i.e. on the wall side. ClaimedBy records the agent that has reserved this point, if any, so two agents don't pile into the same spot.
