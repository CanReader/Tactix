template<std::size_t ChannelCount>
void Tactix::FTactixInfluenceMap< ChannelCount >::Stamp ( std::size_t Channel,
FTactixVec2 World,
float Radius,
float Value )
inline

Adds a circular, linearly-falling-off contribution to a channel.

Each affected cell gains Value * max(0, 1 - distance/Radius), so the peak is at World and it fades to nothing at Radius. Stamps accumulate; use Decay or Clear to drain them.

Parameters
ChannelChannel index; out-of-range or an empty map is a no-op.
WorldWorld-space center of the stamp.
RadiusFalloff radius in world units; non-positive is a no-op.
ValuePeak contribution at the center (may be negative).
