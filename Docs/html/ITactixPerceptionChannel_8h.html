Detailed Description

Stimulus record and channel interface for the perception memory.

A "channel" is a kind of stimulus: sight, sound, damage, radio, smell, and so on. The perception memory itself doesn't care what a channel means; it just stores stimuli tagged with a channel ID and fades them according to that channel's half-life. Projects extend the set by implementing ITactixPerceptionChannel with a fresh ChannelId and half-life.
