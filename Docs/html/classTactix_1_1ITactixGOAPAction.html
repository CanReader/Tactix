Detailed Description

One operator the GOAP planner can chain to reach a goal.

In the search, an action is a transition: it applies when the current state satisfies its preconditions, and following it overlays its effects onto the state at the given cost. Preconditions/effects/cost are queried fresh against the agent context, so the same action object can behave differently per agent.
