float Floor = 0.05f )

Ages every stored stimulus by half-life decay.

Scales each stimulus by 2^(-age / half_life) and zeroes any that drop below Floor. Stimuli already at zero, or whose timestamp is in the future relative to NowSeconds, are skipped.

Parameters
NowSecondsCurrent game time in seconds.
DefaultHalfLifeHalf-life applied to all stimuli here, in seconds. This call decays uniformly; per-channel half-lives are applied by decaying with the relevant channel's value. A non-positive value is clamped up to 1 to avoid dividing by zero.
FloorIntensity below which a stimulus is treated as gone.
