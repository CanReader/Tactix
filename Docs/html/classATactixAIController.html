◆ GetScratchArena()

Tactix::FTactixArena & ATactixAIController::GetScratchArena ( )

The scratch arena used for plan allocation.

Returns
Reference to this controller's arena.
Warning
Call Tactix::FTactixArena::Reset before each planning pass. Plans returned by the planners point into this arena, so resetting it invalidates the previous plan. Don't reset while a plan from it is still being executed.
