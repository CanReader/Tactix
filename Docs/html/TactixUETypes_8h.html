Detailed Description

Blueprint-exposed mirrors of Tactix enums and config structs.

The engine-agnostic modules can't carry UENUM / UPROPERTY metadata, because they're forbidden from including UE headers. So the designer-facing variants live here, in TactixUE, with a "BP" suffix where they shadow a core enum. Conversion between these and the real Tactix types happens inside this module (for example in UTactixUtilityAsset); the core never sees this file.
