Detailed Description

Owns the schedule and parcels out which agents may tick each frame.

Single-threaded: the owning AI controller/subsystem drives it from the game thread. Not safe to call concurrently.

std::size_t Tactix::FTactixAgentScheduler::Collect ( double NowSeconds,
uint32_t FrameBudgetUs,
FTactixHandle< ITactixAgent > * OutAgents,
std::size_t OutCapacity )

Collects the agents that should tick this frame.

Pops agents in due order while their NextTickTime has elapsed, summing expected costs and stopping once the next agent would push the total past FrameBudgetUs. The first due agent is always allowed through even if it alone exceeds the budget, otherwise a single expensive agent could be starved forever. Each collected agent is immediately rescheduled at NowSeconds + TickIntervalMs/1000, so the caller can just loop over the results and invoke each agent's tick.

Parameters
NowSecondsCurrent game time in seconds.
FrameBudgetUsSoft CPU ceiling for this frame's AI, in microseconds.
OutAgentsCaller-owned buffer that receives the due agents.
OutCapacitySize of OutAgents; collection also stops when full.
Returns
Number of agents written to OutAgents.
