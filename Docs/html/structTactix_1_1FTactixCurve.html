Detailed Description

A plain-data response curve: a shape plus its tuning parameters.

Trivially copyable, cheap to pass by value, and safe to bake into data assets. Build one with the Make* factories rather than filling the fields by hand so unused parameters (e.g. Exp for a linear curve) stay at their neutral values.

float Tactix::FTactixCurve::Evaluate ( float X) const
noexcept

Evaluates the curve at X.

Dispatches on Type and clamps the result into [0, 1].

Parameters
XRaw consideration input. There's no required range; the shape and the final clamp decide what comes out.
Returns
A score in [0, 1]. A Custom curve with a null CustomFn returns 0.

The clamp via Saturate is applied to every branch, including Logistic which already lives in (0, 1), because extreme Slope values can push a float a hair past 1.0 through rounding. Quadratic needs special care: std::pow of a negative base with a fractional exponent is NaN, so we power the magnitude and copy the sign back, which keeps the curve sensible for inputs below Shift.
