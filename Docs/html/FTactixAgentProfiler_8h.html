Detailed Description

Per-agent timing recorder for spotting which agents and which phases cost the most.

Samples are tagged by agent and by a context name ("GOAPPlan", "HTNPlan", "UtilityEval", ...) and kept in a fixed ring, so old data ages out on its own. Query worst-case and average microseconds per agent/context, or dump a top-N report. The easy way to feed it is the RAII FTactixTimerScope:

{
FTactixTimerScope Timer(AgentHandle, "GOAPPlan");
FTactixGOAPPlanner::Plan(...);
} // elapsed recorded when Timer leaves scope
