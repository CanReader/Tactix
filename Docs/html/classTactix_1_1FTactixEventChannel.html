Detailed Description

template<typename EventT, std::size_t MaxSubscribers = 16>
class Tactix::FTactixEventChannel< EventT, MaxSubscribers >

A single-event publish/subscribe channel with fixed capacity.

Template Parameters
EventTPayload type, passed to handlers by const reference.
MaxSubscribersLargest number of concurrent subscribers. Past this, Subscribe starts refusing.

Handler signature. The void* is the context given to Subscribe.

A free function or static method is required, not a capturing lambda, which is the price of staying allocation-free. Carry per-instance state through the context pointer (typically this).

Parameters

Returns

Note
Subscribing the same (Fn, Ctx) twice registers it twice, so it will fire twice. Unsubscribe removes one of them.
