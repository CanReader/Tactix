Detailed Description

Unreal-facing module object for TactixSystems (L3 behaviour systems).

This module is the home of the actual AI: Utility selection, the GOAP and HTN planners, the cover registry, the influence map, and squad/formation logic. All of it is engine-agnostic and depends only on TactixCore, so most of the surface lives in headers under Utility/, GOAP/, HTN/, Squad/, Cover/ and Spatial/. This file only carries the IModuleInterface object Unreal needs to load and unload the module.
