Detailed Description

Capacity-independent interface to a blackboard.

FTactixBlackboard is templated on capacity, which would otherwise leak into every signature that wants to touch one. This abstract base hides that: an FTactixAgentContext, a BT service, anything, can hold a FTactixBlackboardRef* without caring how big the concrete table is.

The typed Get / Set templates dispatch to the per-type virtuals via if constexpr, so callers write bb.Set(key, value) and the right virtual is chosen at compile time.
