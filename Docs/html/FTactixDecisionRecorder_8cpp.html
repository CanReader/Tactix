Detailed Description

Implements the decision ring buffer: recording, history/all queries, and the log dump.

Record stamps each entry with wall time. The query helpers do the ring index arithmetic to read newest-first (history, dump) or oldest-first (getAll), with the extra + kRingSize terms keeping the modulo positive.
