Runs A* from Start to Goal over the given actions.

Every allocation, node pool, open-set heap, closed-set table, and the returned step array, comes out of Scratch, so the planner itself never touches the heap. The search is admissible (uses FTactixWorldState::Distance), so the first plan found is optimal in total cost.

Parameters
StartCurrent world state.
GoalDesired facts (its mask says which matter).
ActionsCandidate actions; null entries are skipped.
ActionCountNumber of entries in Actions.
CtxAgent context, forwarded to every action query.
ScratchArena for all working memory and the result's steps.
ConfigSearch limits.
Returns
A plan. bValid is false if Goal is unreachable within the limits or the arena runs out of space. A valid plan with Count == 0 means Start already satisfies Goal.
