Detailed Description

Symbolic world state for the GOAP planner: 64 boolean facts plus a "defined" mask.

The planner reasons over symbolic facts ("HasAmmo", "InCover", "EnemyVisible"), each one a bit addressed by index. Two 64-bit words carry everything: Values holds the truth of each fact and Mask records which facts this state even has an opinion about. A bit that's unset in Mask is "don't care", which is how a goal can require just two facts while ignoring the other 62.

This is intentionally booleans only. Float or scalar predicates (distances, thresholds) don't belong here; they're handled in ITactixGOAPAction via the agent context, leaving the planner's graph small and hashable.
